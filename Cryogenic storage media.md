# Cryogenic storage media protocol
This media is used to store bacterial culture at -80C
***Per 250mL***

| Ingredient | Amount |
|------------|--------|
| Nutrient broth | 2 g |
| Yeast Extract | 0.5 g |
| Glucose | 1.25 g |
| Glycerol | 37.5 g |
| Distilled water | add to make final volume 250 mL |

# Reference
Burr, T.J., Matteson, M.C., Smith, C.A., Corral-Garcia, M.R. and Huang, T.C., 1996. Effectiveness of bacteria and yeasts from apple orchards as biological control agents of apple scab. Biological Control, 6(2), pp.151-157.
