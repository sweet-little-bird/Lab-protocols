# Luria-Bertani agar media protocol
***Per 500mL***
***Do not add agar if making broth***

| Ingredient | Amount |
|------------|--------|
| Tryptone | 5 g |
| Yeast extract | 2.5 g |
| NaCl | 5 g |
| Agar | 7.5 g |
| Distilled water | add to make final volume 500 mL |

Miller LB is 10g NaCl per litre and Kennox LB is 5g NaCl per litre.
