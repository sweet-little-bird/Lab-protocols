# Super Optimal broth with Catabolite repression (SOC) protocol
***Per 1L***


| Ingredient | Amount |
|------------|--------|
| Casein peptone (Tryptone) | 20 g |
| Yeast extract | 5 g |
| NaCl | 0.5845 g |
| KCl | 0.184 g |
| MgCl2 | 0.9521 g |
| anhy. MgSO4 | 1.2036 g |  ***Use 2.465g if using MgSO4.7H2O***
| Glucose | 3.6032 g |
| Distilled water | add to make final volume 500 mL |

This medium is very sensitive to contaminatiom, so making small aliquots for autoclaving and a single time use is recommended.
