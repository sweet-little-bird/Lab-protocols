# Sucrose Peptone Agar medium (SPA) protocol
***Per 500mL***

***Do not add agar if making broth***

| Ingredient | Amount |
|------------|--------|
| Sucrose | 10 g |
| Peptone | 2.5 g |
| K2HPO4 | 0.75 g |
| MgSO4.7H2O | 0.125 g |
| Agar | 7.5 g |
| Distilled water | add to make final volume 500 mL |
